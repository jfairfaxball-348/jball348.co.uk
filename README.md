# Personal Brand Website

A personal brand website built with Next.js, TypeScript, Tailwind CSS, GitHub, and Vercel.

## Goals

* Professional online presence
* Project portfolio
* Content platform
* Long-term personal brand hub

## Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* GitHub
* Vercel

## Repository Structure

app/
components/
content/
public/
lib/

## Development Workflow

1. Plan feature
2. Update PROJECT_CONTEXT.md if required
3. Implement change
4. Commit to GitHub
5. Vercel deploys automatically
6. Review live site
