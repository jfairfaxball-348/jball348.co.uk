The visual style should feel:

* Modern
* Professional
* Technical
* Confident
* Minimal

Inspirations:

* Vercel
* Linear
* OpenAI
* Anthropic

Dark mode is the default experience.

Avoid:

* Flashy animations
* Excessive colour
* Complex visual effects
* Unnecessary dependencies
